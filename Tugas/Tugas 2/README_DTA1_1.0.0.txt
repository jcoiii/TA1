Template Laporan Tugas Akhir I ini dibuat oleh Dr. Ir. Ali Sadiyoko, M.T. untuk membakukan format penulisan Draft Laporan Tugas Akhir di Program Studi Sarjana Teknik Elektro (Konsentrasi Mekatronika), Fakultas Teknologi Industri, Universitas Katolik Parahyangan.
Draft Laporan Tugas Akhir I ini dibuat sebagai laporan proses pengerjaan Tugas Akhir I (IME 184400), digunakan sebagai bahan rujukan penilaian bagi Dosen Pembimbing dan Dosen Penguji pada saat Sidang Tugas Akhir I.
 
Versi : 1.0.0
Tanggal : 20 Februari 2019
Format Standar Draft TA :
1. Ukuran kertas A4, satu sisi.
2. Margin, atas= 3cm, bawah= 3cm, kiri= 4cm, kanan= 3cm.
3. Bentuk huruf = Roman, 12 pt (LaTeX).
2. Identitas penulis laporan/ peserta Tugas Akhir I, judul laporan dan dosen pembimbing dta1-ident.tex.




FILE-FILE TERDIRI DARI :
------------------------
1. dta1-main.tex (file utama, file induk)
   
2. dta1-preamble.tex (file yang menyimpan format-format, deklarasi package, definition, dll, yg ribet2)
   
3. dta1-sampul.tex
   
4. dta1-sampuldalam.tex
   
5. dta1-pengesahan.tex
   
6. dta1-daftarisi.tex
  (file untuk Daftar Isi dokumen ini)
  
7. dta1-daftargbr.tex
 (file untuk Daftar Gambar)
   
8. dta1-daftpust.tex (file untuk Daftar Pustaka)
   
9. dta1-daftlamp.tex (file untuk Daftar Lampiran, jika diperlukan)
 
10. dta1-dafttbl.tex (file untuk Daftar Tabel)
 
11. dta1-ident.tex 
(file untuk diisi identitas penulis laporan draft TA, judul laporan dan dosen pembimbing)
    
12. dta1-abstrak.tex
 (file untuk Abstrak)
   
13. dta1-pengantar.tex
 (file untuk Kata Pengantar)
   
14. dta1-daftsimbol.tex
 (file untuk Daftar Simbol)
  
15. bab1.tex 
(file untuk BAB I: PENDAHULUAN)
   
16. bab2.tex (file untuk BAB II: KAJIAN PUSTAKA)
   
17. bab3.tex 
(file untuk BAB III: DISAIN/ PERANCANGAN SISTEM)   
20. pustaka.bib (daftar pustaka yang digunakan)

FITUR:
1. Caption pada judul gambar dan tabel menggunakan titik (.)
2. Babel package with [indonesian] option, for Bahasa Indonesia.
3. Judul awal bab 'Bab' menjadi 'BAB' dan nama bab menggunakan huruf KAPITAL semua.
4. Ada jarak 15 mm di antara nama bab dengan alinea pertama tiap bab.
